# Tailwind

Tailwind 是一个 CSS 框架，它为我们提供了单一用途的utility classes， which are
opinionated for the most part, 它可以帮助我们在markup或 .js.jsx.ts.tsx 文件中设计我们的网页。 [网站Tailwind](https://tailwindcss.com/)

## 使用

您可以使用以下 Qwik 启动脚本轻松添加 Tailwind：

```bash
npm run qwik add tailwind
```

前面的命令会更新您的应用并在 `vite.config.ts` 中设置正确的配置。

它还将新文件添加到您的 `src` 文件夹中。

- `postcss.config.js` 
- `tailwind.config.js`  

## Action Required

After execute command success you need add the next lines
in your `global.css`

```css

# global.css file

@tailwind base;
@tailwind components;
@tailwind utilities;

...stuff...

```
